# Example code for realtime webapp with Node.JS

## How to run

### install depdendency


```
npm i
```


```
# or with yarn
yarn 
```

### run project

```
npm start 

# or

yarn start
```

as default 
you can see in result in each

- http://localhost:3002  
- http://localhost:3003 
- http://localhost:3004
- http://localhost:3005
